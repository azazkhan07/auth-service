# Auth Service

Authentication and Authorization Microservice for the PayFlow Fintech Platform.

## Features

- User Registration
- User Login
- JWT Token Generation
- Token Validation
- Password Encryption (BCrypt)
- Role Based Access
- Spring Security Integration
- OpenFeign Support
- Swagger Documentation

## Tech Stack

- Java 21
- Spring Boot 3
- Spring Security
- JWT
- PostgreSQL
- Spring Cloud OpenFeign
- Eureka Client
- Maven
